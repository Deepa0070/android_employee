## 🚀 About The Project

**Employee Management App** is a native Android application built with Java that simplifies employee data handling for businesses of all sizes. Whether you're managing a small team or a large organization, this app brings all your employee records to your fingertips.

> Built with simplicity and performance in mind — no clutter, just what you need.

---

## ✨ Features

- 📋 **Add Employees** — Quickly register new employees with all essential details
- 👁️ **View Records** — Browse through a clean, organized list of all employees
- ✏️ **Edit Information** — Update employee details anytime with ease
- 🗑️ **Delete Records** — Remove employees cleanly from the database
- 🔍 **Search & Filter** — Find any employee in seconds
- 💾 **Local Storage** — All data stored securely on-device

## ⚡ Getting Started

### Prerequisites

- Android Studio (Hedgehog or later)
- Android SDK 21+
- Java 8+

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Deepa0070/android_employee.git
   ```

2. **Open in Android Studio**
   - Launch Android Studio
   - Select **Open an Existing Project**
   - Navigate to the cloned folder

3. **Build & Run**
   - Let Gradle sync complete
   - Click ▶️ **Run** or press `Shift + F10`
   - Choose your emulator or connected device
